## ğŸ“Œ Project Overview

A **real-time Credit Risk Prediction Dashboard & App** built using **Python, scikit-learn, and Streamlit**.
It predicts whether a customer is **high risk** or **low risk** for credit default, based on features like **age, income, debt ratio, transaction history, employment type, education level**, and more.

The project uses a **synthetic but realistic credit dataset** with **numerical and categorical features**, ideal for **imbalanced classification**, **feature importance analysis**, and **interactive visualizations**.

---

## ğŸ”¹ Features

* Real-time **Credit Risk Prediction** using pre-trained ML model
* Input customer details via **interactive Streamlit forms**
* Encodes categorical features and scales numeric features automatically
* Displays **prediction class** (High/Low Risk) and **probability**
* Modern **interactive dashboard** for exploring customer data
* Save & load trained models, encoders, and scalers (`.pkl` files)
* Open-source for educational & research purposes
